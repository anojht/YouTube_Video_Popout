Microsoft Edge Instructions--

1) In the address bar in Edge type about:flags
2) Tick box that says enable developer mode for edge extensions
3) After restarting browser click on the more button (...) and click extensions
4) Click load extension and select the downloaded folder
5) Click on the more button (...) and right click on extension icon and click show next to address bar
6) Enjoy!
